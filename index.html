Weekly Objectives

By the end of this module, you will be able to:

1.1 Explain the differences between scripting and programming languages.

1.2 Identify various types of scripting and programming languages.

1.3 Demonstrate a basic knowledge of modular scripting and programming concepts, theories, and processes.

1.4 Identify and describe the various scripting concepts and techniques that govern the science of interactive design.
